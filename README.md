````markdown
    git clone [https://github.com/aneeshsrinivas/aneeshverse.git](https://github.com/aneeshsrinivas/aneeshverse.git)
    ```
2.  **Navigate to the project directory:**
    ```sh
    cd aneeshverse
    ```
3.  **Install dependencies:**
    ```sh
    npm install
    # or
    yarn install
    ```
````
